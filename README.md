# Sparks of the Past

This campaign repository is managed by CampaignRepo.

## Structure

- Pages live in `/wiki/pages`.
- Media lives in `/wiki/media`.
- Templates live in `/wiki/templates`.
- Character imports live in `/wiki/imports/characters`.
- The portable search snapshot lives in `/wiki/search/index.json`.
- Campaign settings live in `/wiki/campaign.yaml`.

Manual edits are welcome. Preserve YAML frontmatter and CampaignRepo conventions for wiki links, visibility, and approvals.

## Links

- GitHub docs: https://docs.github.com/repositories/creating-and-managing-repositories/creating-a-new-repository
- CampaignRepo wiki links: use `[[Page Name]]` or `[[Page Name|label]]`.
- GM-only sections: wrap secret content in `:::gm` blocks.
